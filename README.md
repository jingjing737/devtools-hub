## 📦 Quick Start

### Installation

```bash
# Clone the repository
git clone https://github.com/jingjing737/devtools-hub.git
cd devtools-hub

# Install dependencies
pip install -r requirements.txt
# or
npm install
```

### Run Dashboard

```bash
# Python
python dashboard/app.py

# Node
node dashboard/app.js
```

Access at: `http://localhost:5000`

## 🛠️ Tools

### Dashboard Features

| Feature | Description |
|---------|-------------|
| System Monitor | CPU, Memory, Disk usage |
| Process Manager | View & manage processes |
| File Browser | Browse project files |
| Git Panel | Git operations UI |
| Log Viewer | Real-time log streaming |
| API Tester | REST API testing |

### CLI Commands

```bash
# System info
devtools sysinfo

# Process management
devtools ps
devtools kill <pid>

# Git helpers
devtools git status
devtools git log

# File operations
devtools ls
devtools find <pattern>
```
